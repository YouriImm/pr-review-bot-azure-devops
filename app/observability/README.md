Setting up Logfire is very easy. Follow the [official documentation](https://logfire.pydantic.dev/docs) to so do.

Alongside the code which configures logfire for the app, this folder contains an importable version of the observability
dashboard of the solution.

Follow the [official logfire documentation](https://logfire.pydantic.dev/docs/guides/web-ui/dashboards/#using-a-standard-dashboard-as-a-template)
for instructions.


**N.B.:** All further documentation relating to observability is in [the docs folder](../../docs/observability.md).
